直接复制以下提示词，发送给你的 AI 助手即可完成安装。

技能介绍

核心功能

  • 实时监控多个 DeFi 协议 - 同时监控 Aave、Compound、Curve 等主流协议的收益率
  • 自动寻找最高 APY - 智能对比收益率，自动切换到最优协议
  • 智能再平衡策略 - 根据市场变化自动调整资产配置
  • Gas 费优化 - 智能选择最佳交易时机，节省 Gas 费用

使用指南

1. 配置环境变量

export WALLET_PRIVATE_KEY="your_private_key" export RPC_URL="https://mainnet.infura.io/v3/YOUR_KEY" export MIN_APY=5.0

2. 启动 Agent

defi-agent start --config config.json # 或使用命令行参数 defi-agent start --protocols aave,compound,curve --min-apy 5.0

3. 监控运行状态

defi-agent status defi-agent logs --tail 100 defi-agent stats --period 24h
