# Web3 Skill Hub

精选 Web3 与 AI Agent 技能市场

## 设计系统

- **风格**: Glassmorphism（毛玻璃效果）
- **配色**: 
  - Primary: #F59E0B (金色)
  - Secondary: #FBBF24 (琥珀色)
  - CTA: #8B5CF6 (紫色)
  - Background: #0F172A (深蓝黑)
  - Text: #F8FAFC (白色)
- **字体**: 
  - 标题: Orbitron (未来科技感)
  - 正文: Exo 2 (现代简洁)

## 功能特性

- 🔍 智能搜索栏（Hero CTA）
- 🎨 8 大技能分类
- ⭐ 精选技能展示
- 🎯 炫酷悬停效果
- 🌊 浮动动画
- 💜 渐变发光效果
- 📱 完全响应式

## 本地预览

```bash
cd /root/.openclaw/workspace/web3-skill-hub
python3 -m http.server 8080
```

然后访问: http://localhost:8080

## 技能分类

1. **DeFi 自动化** - 收益聚合、流动性管理
2. **合约审计** - 安全扫描、漏洞检测
3. **交易机器人** - MEV、套利、狙击
4. **链上分析** - Smart Money、巨鲸追踪
5. **NFT 工具** - 批量铸造、扫地板
6. **空投猎人** - 自动领取、多账号管理
7. **智能合约** - 部署、交互、监控
8. **社交代币** - Farcaster、Lens Protocol
